== README

Installation

1. Download the files and unzip it.
2. Open the project (RubyMine recommended)
3. In the console window of rubymine write "bundle install" to install all the gems
4. Then write "rake db:migrate" to install the database.
5. Then write "rake db:seed" to create data
6. When you test the application upload the "Postman.json.txt" found in the root of the project to postman

Don't forget to run the "post user auth" and replace the userkey code in the header.
